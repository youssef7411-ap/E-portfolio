# E-Portfolio Redesign Plan

## Project Overview
Redesign the e-portfolio website with:
1. Subject cover images with admin upload capability
2. Bright, clean, modern homepage with infographic-style elements
3. Subject pages with cover image banners

---

- The `image` field already exists in `backend/src/models/Subject.js`

## Phase 3: Homepage Redesign

### 3.1 Design System Update
**File**: `frontend/src/styles/index.css`

**New color palette (Bright & Clean)**:
```css
:root {
  /* Light background - main change */
  --bg:            #fafbfc;
  --bg-glass:      rgba(255, 255, 255, 0.85);
  --bg-2:          #f3f4f6;
  --bg-3:          #e5e7eb;
  
  /* Dark text for contrast */
  --text:          #111827;
  --text-2:        #374151;
  --text-3:        #6b7280;
  
  /* Border */
  --border:        rgba(0, 0, 0, 0.08);
```

## Design Principles Summary

| Element | Approach |
|---------|----------|
| Background | Light (#fafbfc) with subtle blob shapes |
| Primary Color | Blue (#3b82f6) |
| Accent | Amber (#f59e0b) for highlights |
| Shapes | Low opacity blobs in background only |
| Handwriting | Section titles only, small usage |
| Cards | White/glass with subtle shadows |
| Typography | Clean sans-serif, clear hierarchy |
| Spacing | Generous padding, clean layout |

---

## Notes

- Keep the existing dark mode toggle functionality
- Ensure all images have fallbacks
- Use CSS-only graphs/visuals (no external libraries)
- Mobile responsive throughout
- Performance: lazy load images, optimize shapes
